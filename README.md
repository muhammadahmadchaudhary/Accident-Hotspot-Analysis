# Accident-Hotspot-Analysis
Tools & Technologies:

Python
Pandas & GeoPandas
Matplotlib
DBSCAN Clustering

Project Highlights:

Loaded and cleaned a dataset with location-based data.
Performed geospatial analysis to convert data into GeoDataFrames.
Visualized accident hotspots using maps of Chicago.
Conducted clustering analysis to identify and visualize accident clusters.
Extracted actionable insights to aid urban planning and safety improvements.

Dataset: https://www.kaggle.com/datasets/nathaniellybrand/chicago-car-crash-dataset
